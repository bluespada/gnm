# gnm
Go Node Manger


## Usages

### List Available Node Version
```bash
gnm list-remote
```

```bash
gnm ls-remote
```

### List Installed Node Version

```bash
gnm list
```

```bash
gnm ls
```

### Use or Switch to Another Node Version

```bash
gnm switch v20.18.1
```

```bash
gnm s v20.18.1
```

### Remove Installed Node

```bash
gnm remove v20.18.1
```

```bash
gnm rm v20.18.1
```

### Uninstall

```bash
gnm destroy
```

```bash
gnm d
```
